# CLAUDE BRAIN — Cérebro Digital do Portal NCFN

Diretório de memória persistente para o assistente Claude Code.
Guarda contexto técnico, decisões arquiteturais, estado de implementações e padrões do projeto.

## Como usar

Ao iniciar uma conversa: ler `ESTADO_ATUAL.md` e `IMPLEMENTACOES.md` para ter contexto completo.
Ao terminar uma implementação: atualizar os arquivos relevantes.
